# Calendar Project

## Description
This project is a dynamic calendar web application that allows users to view and add events to specific dates. It features navigation between months, input validation for dates and times, and a responsive design. The application is built using HTML, CSS, and JavaScript, employing various modern web development concepts.

## Files
- **index.html**: The main HTML file that structures the web page.
- **css/calendar.css**: The CSS file that styles the calendar.
- **js/calendar.js**: The main JavaScript file that controls the calendar functionality.
- **js/calendarSetup.js**: JavaScript module for setting up the calendar.
- **js/findDates.js**: JavaScript module for navigating between months.
- **js/events.js**: JavaScript module for handling events.

## Concepts Used
- **Functions**: Modular and reusable code blocks.
- **Control Flow**: Conditional statements and loops to control the logic.
- **Arrays**: Managing lists of dates and events.
- **Scope & Hoisting**: Managing variable scope and hoisting.
- **Events & Event Loop**: Handling user interactions and asynchronous operations.
- **ES6**: Modern JavaScript features like arrow functions, template literals, and modules.
- **DOM Manipulation**: Interacting with and modifying the HTML document.
- **Object-Oriented Programming (OOP)**: Using objects to represent events and dates.
- **Objects**: Storing data and functionalities.
- **Loops**: Iterating over arrays and objects.
- **Regular Expressions**: Validating input formats for dates and times.
- **Responsive Design**: Ensuring the application works well on various screen sizes.
- **CSS Variables**: Using custom properties for styling.
- **Event Listeners**: Responding to user inputs like clicks and keypresses.
- **Date and Time Manipulation**: Handling and formatting dates and times.

## How to Run
1. Clone or download the project files.
2. Open `index.html` in a web browser to view the calendar application.
